Project Description:

This is a Project to Harvest Data from YouTube using Google API Key with Python and stored to MySQL Tables with Python-MySQL connector ,  The Data is Scraped using Channel ID , The Harvested data is of various IDs are stored in different Tables like Playlist ,Video and Comments with all its own IDs as Primary Key values and Later Quarried in MySQL to Join Operation to show the results of various tables combined and shown on a Streamlit Application .


Tools Used :
  1. Python 
  2. Google API Key
  3. MySQL
  4. Streamlit


Requirements:
